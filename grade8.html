Lesson 1 • Linear Equations & Inequalities

Real Numbers & Properties

NY State Standard: 8.NS.1

📌 Learning Objective

Students will classify numbers as rational or irrational and understand the structure of the real number system.

The Real Number System

Rational numbers can be written as a fraction a/b where a and b are integers (b≠0). This includes integers, fractions, terminating decimals, and repeating decimals.

Irrational numbers CANNOT be written as a fraction. Their decimals go on forever without repeating.

Examples: √2 = 1.41421356..., π = 3.14159265...

Hierarchy: Natural ⊂ Whole ⊂ Integer ⊂ Rational ⊂ Real. Irrational numbers are also Real but NOT Rational.

Key test: Can it be written as a/b? If yes → rational. If no → irrational.

The Number System Hierarchy

Natural numbers: 1, 2, 3, ... (counting numbers)

Whole numbers: 0, 1, 2, 3, ... (naturals + zero)

Integers: ..., -2, -1, 0, 1, 2, ... (whole + negatives)

Rational: Any number that can be written as a/b (includes all above + fractions + terminating/repeating decimals)

Irrational: Cannot be written as a/b. Decimals never end and never repeat. Examples: √2, √3, π, e.

Real numbers: ALL of the above combined. Every point on the number line is a real number.

Common Mistakes

√4 = 2, which IS rational. Just because a number has a radical does not make it irrational — check if it simplifies to a whole number!

0.101001000100001... LOOKS like it has a pattern, but the pattern never repeats the same block. This is irrational.

📝 Worked Example: Classifying Numbers

Classify each: 0.75, √5, -3, π, 0.333...

  1. 0.75 = 3/4 → rational (terminating decimal).
  2. √5 = 2.2360679... → irrational (non-repeating, non-terminating).
  3. -3 = -3/1 → rational (integer).
  4. π = 3.14159... → irrational.
  5. 0.333... = 1/3 → rational (repeating decimal).
💡 Remember: If the decimal terminates or repeats, it is rational. If it goes on forever with no pattern, it is irrational!

🤝 Guided Practice

Example 1: Is √9 rational or irrational?

√9 = 3. Rational

Example 2: Is √7 rational or irrational?

√7 = 2.6457... Irrational

Example 3: Is 0.1010010001... rational?

No pattern repeats. Irrational

Example 4: Is 0 rational or irrational?

0 = 0/1. Rational (it is an integer)

Example 5: Is √100 rational?

√100 = 10. Rational (perfect square)

Lesson 2 • Linear Equations & Inequalities

Operations with Irrational Numbers

NY State Standard: 8.NS.2

📌 Learning Objective

Students will approximate irrational numbers and locate them on a number line.

Approximating Irrational Numbers

To approximate √n, find which two perfect squares n falls between.

Example: √20. Since 4²=16 and 5²=25, √20 is between 4 and 5.

20 is closer to 16 than 25, so √20 ≈ 4.5 (actually 4.47...).

Operations: You can add, subtract, multiply with irrational numbers:

3√2 + 5√2 = 8√2 (combine like terms, same as 3x + 5x = 8x).

√2 × √3 = √6.    But √2 + √3 ≠ √5!

Simplifying Square Roots

To simplify √n, look for perfect square factors.

√50 = √(25 × 2) = √25 × √2 = 5√2.

√72 = √(36 × 2) = 6√2.

Rules for Radical Operations

Can combine: 3√5 + 7√5 = 10√5 (like terms, same radicand).

Can multiply: √a × √b = √(ab). So √3 × √12 = √36 = 6.

Cannot add different radicands: √2 + √3 ≠ √5. There is no shortcut for adding different square roots!

📝 Worked Example: Estimating

Between which two integers is √50?

  1. Perfect squares: 7²=49, 8²=64.
  2. √50 is between 7 and 8.
  3. 50 is very close to 49, so √50 ≈ 7.07.
💡 Remember: Find the two perfect squares on either side. The closer n is to one, the closer the root is to that integer!

🤝 Guided Practice

Example 1: √30 is between which integers?

5²=25, 6²=36. Between 5 and 6

Example 2: 2√3 + 4√3 = ?

6√3 (combine like radicals)

Example 3: √4 × √9 = ?

√36 = 6

Example 4: Simplify √18.

√(9×2) = 3√2

Example 5: √5 × √5 = ?

√25 = 5

Lesson 3 • Linear Equations & Inequalities

Solving Linear Equations

NY State Standard: 8.EE.7

📌 Learning Objective

Students will solve multi-step linear equations with rational coefficients, including variables on both sides.

Solving Multi-Step Equations

Steps:

1. Distribute to clear parentheses.

2. Combine like terms on each side.

3. Move variables to one side (add/subtract).

4. Move constants to the other side.

5. Divide to isolate x.

Example: 3(x + 2) = 2x + 10 → 3x + 6 = 2x + 10 → x + 6 = 10 → x = 4.

Equations with Variables on Both Sides

When x appears on BOTH sides, move all x-terms to one side first.

5x + 3 = 3x + 11 → 2x + 3 = 11 → 2x = 8 → x = 4.

Equations with Fractions

Multiply every term by the LCD to clear fractions.

x/3 + x/4 = 7. LCD = 12: 4x + 3x = 84 → 7x = 84 → x = 12.

Always Check

Plug your answer back into the ORIGINAL equation to verify both sides are equal.

📝 Worked Example: Multi-Step

Solve: 5x - 3(x + 2) = 4x - 12

  1. Distribute: 5x - 3x - 6 = 4x - 12.
  2. Combine: 2x - 6 = 4x - 12.
  3. Move variables: -6 = 2x - 12.
  4. Move constants: 6 = 2x.
  5. Divide: x = 3.
💡 Remember: Distribute first, combine like terms, then get all x-terms on one side and constants on the other!

🤝 Guided Practice

Example 1: 2(x-4) = 10?

2x-8=10, 2x=18, x=9

Example 2: 4x + 5 = 2x + 13?

2x=8, x=4

Example 3: x/3 + 2 = 5?

x/3=3, x=9

Example 4: 6x - 4 = 2(x + 6)?

6x-4=2x+12, 4x=16, x=4

Example 5: x/2 + 5 = 9?

x/2=4, x=8

Lesson 4 • Linear Equations & Inequalities

Equations with Special Cases

NY State Standard: 8.EE.7a

📌 Learning Objective

Students will recognize equations with no solution or infinitely many solutions.

Special Cases in Equations

One solution: You get x = a number. Example: 2x = 10 → x = 5.

No solution: Variables cancel and you get a false statement. Example: 2x + 3 = 2x + 7 → 3 = 7 (false!). No value of x works.

Infinitely many solutions: Variables cancel and you get a true statement. Example: 3(x+1) = 3x + 3 → 3x+3 = 3x+3 → 0 = 0 (always true!). Every x works.

How to tell: Solve normally. If the variables disappear, check if what remains is true or false.

How to Recognize Special Cases

Solve normally. If the variables cancel out:

True statement (like 5=5 or 0=0): ALL real numbers are solutions → infinitely many.

False statement (like 3=7 or 0=5): NO value works → no solution.

If a variable remains, there is exactly one solution.

📝 Worked Example: Identifying Special Cases

Solve: 4(x+2) = 4x + 5

  1. Distribute: 4x + 8 = 4x + 5.
  2. Subtract 4x: 8 = 5.
  3. 8 ≠ 5. This is false.
  4. Answer: No solution.
💡 Remember: If variables cancel and you get a TRUE statement = infinite solutions. FALSE statement = no solution!

🤝 Guided Practice

Example 1: 3x + 1 = 3x + 1?

0=0, true. Infinitely many solutions

Example 2: x + 5 = x - 2?

5=-2, false. No solution

Example 3: 2x + 4 = 10?

2x=6, x=3. One solution

Example 4: 2(x+3) = 2x + 6?

2x+6=2x+6. 0=0. Infinite solutions

Example 5: x + 1 = x + 2?

1=2. False. No solution

Lesson 5 • Linear Equations & Inequalities

Literal Equations

NY State Standard: 8.EE.7

📌 Learning Objective

Students will rearrange formulas and literal equations to solve for a specified variable.

Literal Equations

A literal equation has multiple variables. You solve for one variable in terms of the others.

Use the same steps as regular equations - just treat other variables like numbers.

Example: Solve A = lw for w. → Divide both sides by l: w = A/l.

Example: Solve y = mx + b for x. → y - b = mx → x = (y-b)/m.

Common formulas: d = rt, A = 1/2 bh, P = 2l + 2w, C = 2πr, I = prt.

Strategy

Treat the variable you are NOT solving for as if it were a number. Use inverse operations to isolate your target variable.

If the target variable is being multiplied, divide. If divided, multiply. If added, subtract.

Important Formulas to Know

d = rt (distance), A = ½bh (triangle), P = 2l + 2w (rectangle perimeter), C = 2πr (circumference), V = lwh (box volume), I = prt (simple interest).

📝 Worked Example: Rearranging

Solve the formula P = 2l + 2w for l.

  1. Subtract 2w: P - 2w = 2l.
  2. Divide by 2: l = (P - 2w)/2.
💡 Remember: Treat other variables like numbers. Use inverse operations to isolate the variable you want!

🤝 Guided Practice

Example 1: Solve d=rt for t.

t = d/r

Example 2: Solve A=1/2 bh for h.

h = 2A/b

Example 3: Solve C=2πr for r.

r = C/(2π)

Example 4: Solve V = lwh for h.

h = V/(lw)

Example 5: Solve y = mx + b for b.

b = y - mx

Lesson 6 • Linear Equations & Inequalities

Solving & Graphing Inequalities

NY State Standard: 8.EE.7

📌 Learning Objective

Students will solve multi-step and compound inequalities and graph their solutions.

Multi-Step & Compound Inequalities

Multi-step: Same as equations. Flip the sign when multiplying/dividing by a negative.

-3x + 5 > 14 → -3x > 9 → x < -3 (flipped!)

Compound inequalities combine two inequalities:

AND: Both must be true. -2 < x < 5 means x is between -2 and 5.

OR: At least one must be true. x < -3 OR x > 4.

AND = intersection (overlap). OR = union (everything from both).

Compound Inequalities on a Number Line

AND graph: Shade the OVERLAP (intersection). Both conditions must be true simultaneously.

OR graph: Shade BOTH regions (union). At least one condition must be true.

AND creates a segment between two values. OR creates two rays going in opposite directions.

📝 Worked Example: Compound Inequality

Solve: -3 ≤ 2x + 1 < 9

  1. Subtract 1 from all three parts: -4 ≤ 2x < 8.
  2. Divide all by 2: -2 ≤ x < 4.
  3. Graph: Closed circle at -2, open circle at 4, shade between.
💡 Remember: For compound inequalities, perform the same operation on ALL parts. Remember to flip signs when dividing by negatives!

🤝 Guided Practice

Example 1: x > 3 AND x < 8?

3 < x < 8 (between 3 and 8)

Example 2: x < -1 OR x > 5?

Two separate regions on the number line

Example 3: -4x ≥ 12?

x ≤ -3 (flip the sign!)

Example 4: x ≥ -2 AND x < 5?

Shade from -2 (closed) to 5 (open)

Example 5: x < 1 OR x > 4?

Two rays: left of 1, right of 4

Lesson 7 • Functions & Graphing

Graphing Linear Functions

NY State Standard: 8.F.3

📌 Learning Objective

Students will graph linear functions and identify slope, y-intercept, x-intercept, and whether the function is increasing or decreasing.

Graphing Linear Functions

A linear function graphs as a straight line. The general form is y = mx + b.

Slope (m): Rise over run. Positive slope goes up-right, negative slope goes down-right.

y-intercept (b): Where the line crosses the y-axis (x=0).

x-intercept: Where the line crosses the x-axis (y=0). Set y=0 and solve.

To graph: 1. Plot the y-intercept. 2. Use slope (rise/run) to find a second point. 3. Draw the line.

Types of Slopes

Positive slope: Line goes up from left to right (increasing). Example: y = 2x + 1.

Negative slope: Line goes down from left to right (decreasing). Example: y = -3x + 5.

Zero slope: Horizontal line. Example: y = 4.

Undefined slope: Vertical line. Example: x = 2. (Not a function!)

Steepness

The larger the absolute value of the slope, the steeper the line. |5| > |2|, so y=5x is steeper than y=2x.

📝 Worked Example: Graphing

Graph y = 2x - 3.

  1. y-intercept: b = -3. Plot (0, -3).
  2. Slope: m = 2 = 2/1. From (0,-3), go up 2, right 1 to (1, -1).
  3. Draw the line through both points.
💡 Remember: Start at the y-intercept, then use slope as rise/run to find the next point!

🤝 Guided Practice

Example 1: y = -x + 4. y-intercept?

(0, 4)

Example 2: y = 3x. Slope?

3

Example 3: y = -2x + 6. x-intercept?

0 = -2x+6, x=3. Point (3,0)

Example 4: Is y = -4x + 2 increasing or decreasing?

Negative slope: decreasing

Example 5: Which is steeper: y = 3x or y = x?

y = 3x (|3| > |1|)

Lesson 8 • Functions & Graphing

Slope-Intercept Form

NY State Standard: 8.F.3

📌 Learning Objective

Students will convert equations to slope-intercept form and use it to graph and analyze linear functions.

Slope-Intercept Form: y = mx + b

Any linear equation can be rewritten in y = mx + b form by solving for y.

Example: 2x + y = 7 → y = -2x + 7. Slope = -2, y-int = 7.

Example: 3x - 2y = 12 → -2y = -3x + 12 → y = (3/2)x - 6. Slope = 3/2, y-int = -6.

Parallel lines have the same slope.

Perpendicular lines have slopes that are negative reciprocals (like 2 and -1/2).

Parallel and Perpendicular Lines

Parallel lines: Same slope, different y-intercepts. They NEVER intersect.

y = 3x + 1 and y = 3x - 4 are parallel (both have slope 3).

Perpendicular lines: Slopes are negative reciprocals (flip and negate).

If one slope is ⅔, the perpendicular slope is -&frac32;. Product = -1.

📝 Worked Example: Converting

Convert 4x - 2y = 8 to slope-intercept form.

  1. Subtract 4x: -2y = -4x + 8.
  2. Divide by -2: y = 2x - 4.
  3. Slope = 2, y-intercept = -4.
💡 Remember: Isolate y by moving everything else to the other side, then divide by the coefficient of y!

🤝 Guided Practice

Example 1: 6x + 3y = 9. Slope?

3y=-6x+9, y=-2x+3. Slope = -2

Example 2: Parallel to y=3x+1?

Same slope: m = 3

Example 3: Perpendicular to y=2x?

Negative reciprocal: m = -1/2

Example 4: Parallel to y = -2x + 5. Slope?

-2 (same slope)

Example 5: Perpendicular to slope 4. Slope?

-¼ (flip and negate)

Lesson 9 • Functions & Graphing

Rate of Change

NY State Standard: 8.F.4

📌 Learning Objective

Students will determine the rate of change from tables, graphs, equations, and real-world situations.

Rate of Change = Slope

The rate of change describes how one quantity changes relative to another. It IS the slope.

From two points: m = (y2 - y1) / (x2 - x1)

From a table: Pick any two rows and calculate Δy/Δx.

From a graph: Pick two clear points and count rise/run.

Interpretation: A rate of change of 3 means y increases by 3 for every 1 unit increase in x.

A negative rate of change means y is decreasing.

Special Slopes

Slope = 0: Horizontal line. y does not change at all.

Slope undefined: Vertical line. x does not change (division by zero in the formula).

Positive slope: y increases as x increases.

Negative slope: y decreases as x increases.

Real-World Rates of Change

Speed = distance/time. Wage = pay/hours. Price = cost/quantity. These are all rates of change!

📝 Worked Example: Finding Slope

Find the slope between points (2, 5) and (6, 13).

  1. Formula: m = (y2-y1)/(x2-x1).
  2. Substitute: m = (13-5)/(6-2) = 8/4.
  3. Slope = 2.
💡 Remember: Slope = rise/run = (change in y)/(change in x). Pick any two points!

🤝 Guided Practice

Example 1: Points (1,3) and (4,9). Slope?

(9-3)/(4-1) = 6/3 = 2

Example 2: Points (0,5) and (2,1). Slope?

(1-5)/(2-0) = -4/2 = -2

Example 3: Slope = 0 means?

Horizontal line (no change in y)

Example 4: Points (3,5) and (3,10). Slope?

(10-5)/(3-3) = 5/0 = undefined (vertical)

Example 5: Cost goes from $40 to $55 as items go from 5 to 8. Rate?

(55-40)/(8-5) = 15/3 = $5 per item

Lesson 10 • Functions & Graphing

Writing Linear Equations

NY State Standard: 8.F.4

📌 Learning Objective

Students will write linear equations given a slope and a point, or given two points.

Writing Linear Equations

Given slope (m) and y-intercept (b): Plug directly into y = mx + b.

Given slope (m) and a point (x1, y1):

Use point-slope form: y - y1 = m(x - x1). Then convert to slope-intercept.

Given two points:

1. Find slope: m = (y2-y1)/(x2-x1).

2. Use one point in y - y1 = m(x - x1).

3. Simplify to y = mx + b.

Point-Slope Form

y - y1 = m(x - x1), where m is the slope and (x1, y1) is any point on the line.

This is useful when you do NOT know the y-intercept but DO know a point and the slope.

After writing point-slope form, simplify to slope-intercept (y = mx + b) by distributing and solving for y.

📝 Worked Example: Two Points

Write the equation through (1, 3) and (3, 7).

  1. Slope: m = (7-3)/(3-1) = 4/2 = 2.
  2. Point-slope: y - 3 = 2(x - 1).
  3. Simplify: y - 3 = 2x - 2 → y = 2x + 1.
💡 Remember: Find slope first, then plug slope and one point into point-slope form, then simplify!

🤝 Guided Practice

Example 1: Slope 3, through (0,5)?

y = 3x + 5

Example 2: Slope -1, through (2,4)?

y-4 = -(x-2), y = -x + 6

Example 3: Through (0,0) and (3,9)?

m=3, y = 3x

Example 4: Slope 5, through (0,0). Equation?

y = 5x (through origin)

Example 5: Through (-1, 3) and (1, 7). Equation?

m=(7-3)/(1-(-1))=2. y-3=2(x+1), y=2x+5

Lesson 11 • Functions & Graphing

Functions

NY State Standard: 8.F.1

📌 Learning Objective

Students will determine if a relation is a function and evaluate functions using function notation.

What Is a Function?

A function assigns exactly one output (y) to each input (x).

Vertical Line Test: If any vertical line crosses the graph more than once, it is NOT a function.

Function notation: f(x) means the output when the input is x. f(3) means plug in x=3.

If f(x) = 2x + 1, then f(3) = 2(3) + 1 = 7.

From a table: If any x-value repeats with different y-values, it is NOT a function.

Function Notation

f(x) is read "f of x." It means the output of function f when the input is x.

f(x) = 2x + 1 means: whatever number you put in for x, double it and add 1.

f(3) = 2(3) + 1 = 7. f(-1) = 2(-1) + 1 = -1.

Domain and Range

Domain: All possible INPUT values (x-values).

Range: All possible OUTPUT values (y-values).

📝 Worked Example: Evaluating Functions

If f(x) = 3x - 5, find f(4).

  1. Substitute x = 4: f(4) = 3(4) - 5.
  2. Calculate: 12 - 5 = 7.
💡 Remember: f(x) just means plug in the value for every x. A function gives exactly one output per input!

🤝 Guided Practice

Example 1: Is {(1,2),(3,4),(1,5)} a function?

No (x=1 maps to 2 AND 5)

Example 2: f(x) = x². f(-3)?

(-3)² = 9

Example 3: Does a circle pass the vertical line test?

No

Example 4: f(x) = x^2 + 1. Find f(0).

0 + 1 = 1

Example 5: Is x^2 + y^2 = 1 a function?

No (circle fails vertical line test)

Lesson 12 • Functions & Graphing

Arithmetic Sequences

NY State Standard: 8.F.3

📌 Learning Objective

Students will write recursive and explicit formulas for arithmetic sequences.

Arithmetic Sequences

An arithmetic sequence has a common difference (d) between consecutive terms.

Example: 3, 7, 11, 15, 19... Common difference d = 4.

Recursive formula: a(n) = a(n-1) + d (next term = previous + d).

Explicit formula: a(n) = a(1) + (n-1)d

For our example: a(n) = 3 + (n-1)(4) = 4n - 1.

The explicit formula lets you find ANY term directly without listing all previous terms.

Connecting Sequences to Functions

An arithmetic sequence IS a linear function where n is the input and a(n) is the output.

a(n) = a(1) + (n-1)d is equivalent to y = d·n + (a(1) - d).

The common difference d is the slope!

📝 Worked Example: Finding Terms

Sequence: 5, 8, 11, 14... Find the 20th term.

  1. First term: a(1) = 5. Common difference: d = 3.
  2. Explicit: a(n) = 5 + (n-1)(3) = 3n + 2.
  3. a(20) = 3(20) + 2 = 62.
💡 Remember: Find d by subtracting consecutive terms. Use a(n) = a(1) + (n-1)d for any term!

Lesson 14 • Systems of Equations

Systems of Equations: Graphing

NY State Standard: 8.EE.8a

📌 Learning Objective

Students will solve systems of two linear equations by graphing and interpret the intersection point.

Solving Systems by Graphing

A system is two (or more) equations. The solution is the point where the lines intersect.

One solution: Lines cross at one point (different slopes).

No solution: Lines are parallel (same slope, different y-intercepts). They never meet.

Infinite solutions: Lines are the same line (same slope AND same y-intercept).

To solve: Graph both lines. Read the intersection point. Check by plugging into BOTH equations.

Types of Solutions

One solution: Different slopes. Lines cross once.

No solution: Same slope, different y-intercepts. Parallel lines never meet.

Infinite solutions: Same slope AND same y-intercept. Same line.

You can tell the number of solutions BEFORE graphing by comparing slopes and y-intercepts!

📝 Worked Example: Graphing a System

Solve by graphing: y = 2x - 1 and y = -x + 5.

  1. Graph line 1: y-int (0,-1), slope 2.
  2. Graph line 2: y-int (0,5), slope -1.
  3. Intersection: (2, 3).
  4. Check: 3=2(2)-1 ✅ and 3=-(2)+5 ✅.
💡 Remember: The solution to a system is the intersection point. Always check your answer in BOTH equations!

Systems of Equations: Substitution

NY State Standard: 8.EE.8b

📌 Learning Objective

Solving by Substitution

Steps:

1. Solve one equation for one variable (get y = ... or x = ...).

2. Substitute that expression into the other equation.

3. Solve for the remaining variable.

4. Back-substitute to find the other variable.

Best when one equation already has a variable isolated (like y = ... or x = ...).

  4. Back-sub: y = 3 + 4 = 7. Solution: (3, 7).
💡 Remember: Substitute the expression for one variable INTO the other equation. Then solve and back-substitute!

📌 Learning Objective

Solving by Elimination

Goal: Add (or subtract) the equations to eliminate one variable.

Steps:

2. If needed, multiply one or both equations so a variable has opposite coefficients.

3. Add the equations to eliminate that variable.

Exponent Rules

NY State Standard: 8.EE.1

📌 Learning Objective

Exponent Rules

Product Rule: x^a · x^b = x^(a+b). Same base? Add exponents.

Quotient Rule: x^a / x^b = x^(a-b). Same base? Subtract exponents.

Power Rule: (x^a)^b = x^(ab). Power to a power? Multiply exponents.

Zero Exponent: x^0 = 1 (anything to the 0 is 1, except 0^0).

Negative Exponent: x^(-n) = 1/x^n. Flip to the denominator.

Power of Product: (xy)^n = x^n · y^n.

Combining Rules

Often you need multiple rules in one problem:

(2x³)² ÷ x&sup4; = 4x&sup6; ÷ x&sup4; = 4x² (power rule, then quotient rule).

Scientific notation uses exponent rules: (3 × 10&sup4;)(2 × 10³) = 6 × 10&sup7;.

📝 Worked Example: Applying Rules

Simplify: (2x^3)^2 · x^4

  1. Power rule: (2x^3)^2 = 2^2 · x^6 = 4x^6.
  2. Product rule: 4x^6 · x^4 = 4x^(6+4) = 4x^10.
💡 Remember: Product = add exponents. Quotient = subtract. Power = multiply. Zero = 1. Negative = reciprocal!

Polynomials: Add & Subtract

NY State Standard: 8.EE.1

📌 Learning Objective

Adding & Subtracting Polynomials

A polynomial is an expression with variables and exponents: 3x^2 + 2x - 5.

Like terms have the same variable AND the same exponent: 3x^2 and 7x^2 are like terms.

Adding: Combine like terms. (3x^2 + 2x) + (5x^2 - x) = 8x^2 + x.

Subtracting: Distribute the negative, then combine.

(4x^2 + 3x) - (2x^2 + x) = 4x^2 + 3x - 2x^2 - x = 2x^2 + 2x.

Classifying Polynomials

By number of terms: Monomial (1 term), Binomial (2 terms), Trinomial (3 terms).

By degree: Linear (degree 1), Quadratic (degree 2), Cubic (degree 3).

Standard form: Write terms in order from highest degree to lowest.

📝 Worked Example: Subtracting Polynomials

Simplify: (5x^2 + 3x - 2) - (2x^2 - x + 4)

  1. Distribute negative: 5x^2 + 3x - 2 - 2x^2 + x - 4.
  2. Combine: (5x^2-2x^2) + (3x+x) + (-2-4).
  3. Answer: 3x^2 + 4x - 6.
💡 Remember: When subtracting, change ALL signs in the second polynomial, then combine like terms!

Multiplying Polynomials

Recognizing these patterns saves time and reduces errors!

  4. Last: 4 · (-3) = -12.

📌 Learning Objective

Factoring

Factoring is the reverse of multiplying. You break an expression into factors.

Step 1: Always check for GCF first!

6x^2 + 9x = 3x(2x + 3). GCF is 3x.

Difference of Two Squares (DOTS):

a^2 - b^2 = (a + b)(a - b)

x^2 - 25 = (x + 5)(x - 5).    4x^2 - 9 = (2x + 3)(2x - 3).

Key: DOTS only works for SUBTRACTION of two perfect squares. x^2 + 25 does NOT factor!

Factoring Strategy

Step 1: Always factor out the GCF first.

Step 2: If 2 terms remain, check for Difference of Two Squares (a² - b²).

Step 3: If 3 terms remain, try trinomial factoring.

Remember: Sum of squares (a² + b²) does NOT factor over the reals!

📝 Worked Example: Factoring DOTS

Factor: 9x^2 - 16

  1. Is it a^2 - b^2? Yes! 9x^2 = (3x)^2, 16 = 4^2.
  2. Apply DOTS: (3x + 4)(3x - 4).
💡 Remember: Always check GCF first! For DOTS: both terms must be perfect squares with subtraction between them.

Graphing Quadratics

A quadratic function y = ax^2 + bx + c graphs as a parabola.

Opens up if a > 0 (U-shape). Opens down if a < 0 (∩-shape).

Vertex: The highest or lowest point. x-coordinate = -b/(2a).

Axis of symmetry: The vertical line x = -b/(2a).

Vertex form: y = a(x-h)^2 + k, where (h,k) is the vertex.

y-intercept: Set x=0 → y = c.

Vertex Form: y = a(x-h)² + k

The vertex is directly readable as (h, k).

Be careful with signs: y = (x-3)² + 2 has vertex (3, 2), NOT (-3, 2).

y = (x+1)² - 4 has vertex (-1, -4) because x+1 = x-(-1).

Converting Between Forms

Standard form → vertex: Use x = -b/(2a) to find vertex x, then plug in for y.

Vertex form → standard: Expand (x-h)² and simplify.

📝 Worked Example: Key Features

  3. Vertex: (3, -4). Opens up (a=1>0).
💡 Remember: Vertex x = -b/(2a). Plug back in to find y. If a>0 opens up, a<0 opens down!

Lesson 25 • Exponential Functions & Statistics

Exponential Functions

NY State Standard: 8.F.3

📌 Learning Objective

Students will model situations with exponential growth and decay functions.

Exponential Growth & Decay

Exponential growth: y = a(1 + r)^t, where a = initial amount, r = rate, t = time.

Exponential decay: y = a(1 - r)^t.

Compound interest: A = P(1 + r/n)^(nt), where P = principal, r = annual rate, n = times compounded per year.

Key difference from linear: Linear grows by a constant AMOUNT. Exponential grows by a constant FACTOR (multiplier).

Example: $500 at 3% annual growth: y = 500(1.03)^t.

Linear vs. Exponential

Linear: Constant RATE of change (add/subtract the same amount each time). y = mx + b.

Exponential: Constant RATIO of change (multiply by the same factor each time). y = a · b^x.

In a table: if y-differences are constant → linear. If y-ratios are constant → exponential.

Exponential growth ALWAYS eventually outpaces linear growth, no matter how steep the linear function is.

📝 Worked Example: Growth Problem

A population of 2000 bacteria doubles every hour. How many after 5 hours?

  1. Formula: y = 2000(2)^t.
  2. At t=5: y = 2000(2^5) = 2000(32).
  3. Answer: 64,000 bacteria.
💡 Remember: Growth = multiply by (1+rate). Decay = multiply by (1-rate). Doubling = multiply by 2!

🤝 Guided Practice

Base > 1 = growth

Example 4: Table: x=0,1,2,3. y=1,3,9,27. Linear or exponential?

Ratios: 3,3,3. Exponential (multiply by 3)

Example 5: y = 500(0.85)^t. Growth or decay?

0.85 < 1. Decay (losing 15% each time)

Geometric Sequences

A geometric sequence has a common ratio (r) between consecutive terms.

Example: 3, 6, 12, 24, 48... r = 2 (each term × 2).

Explicit formula: a(n) = a(1) · r^(n-1)

For our example: a(n) = 3 · 2^(n-1).

Arithmetic vs Geometric:

Arithmetic: common DIFFERENCE (add same number). Geometric: common RATIO (multiply same number).

Geometric sequences ARE exponential functions!

Arithmetic vs Geometric Summary

Arithmetic: Add a common difference d. Formula: a(n) = a(1) + (n-1)d. Graphs as a LINE.

Geometric: Multiply by a common ratio r. Formula: a(n) = a(1) · r^(n-1). Graphs as a CURVE.

If r > 1: exponential growth. If 0 < r < 1: exponential decay.

📝 Worked Example: Geometric Sequence

Sequence: 5, 15, 45, 135... Find the 6th term.

  1. Common ratio: r = 15/5 = 3.
  2. Formula: a(n) = 5 · 3^(n-1).
  3. a(6) = 5 · 3^5 = 5 · 243 = 1,215.
💡 Remember: Find r by dividing any term by the previous term. Then use a(n) = a(1) · r^(n-1)!

🤝 Guided Practice

5

Example 2: a(1)=4, r=3. 4th term?

4(3^3) = 4(27) = 108

Example 3: 100, 50, 25... r = ?

1/2 (halving)

Example 4: 1, -3, 9, -27... Geometric? What is r?

r = -3. Yes, geometric (alternating signs)

Example 5: a(1)=100, r=0.5. What happens to the terms?

They get smaller and smaller (decay)

Scatter Plots & Line of Best Fit

NY State Standard: 8.SP.1

📌 Learning Objective

Students will construct scatter plots, draw lines of best fit, and interpret correlation.

Scatter Plots & Correlation

A scatter plot shows the relationship between two quantitative variables.

Positive correlation: As x increases, y increases (upward trend).

Negative correlation: As x increases, y decreases (downward trend).

No correlation: No clear pattern.

Line of best fit: A straight line that best represents the data trend. About half the points should be above and half below.

Use the line to make predictions (interpolation within data, extrapolation beyond).

Correlation Coefficient (r)

The correlation coefficient r measures the strength and direction of a linear relationship.

r close to +1: Strong positive correlation.

r close to -1: Strong negative correlation.

r close to 0: Weak or no linear correlation.

|r| > 0.8 is considered strong. |r| between 0.5 and 0.8 is moderate. |r| < 0.5 is weak.

📝 Worked Example: Interpreting

Hours studied: 1,2,3,4,5. Test scores: 55,62,70,75,85. Describe the correlation.

  1. Pattern: As hours increase, scores increase.
  2. Correlation: Positive (upward trend).
  3. Prediction: A student studying 6 hours might score around 90.
💡 Remember: Positive = both go up. Negative = one goes up, other goes down. No pattern = no correlation!

Two-Way Tables & Statistics

NY State Standard: 8.SP.4

📌 Learning Objective

Students will construct and interpret two-way frequency tables and find relative frequencies.

Two-Way Frequency Tables

A two-way table displays data for two categorical variables.

Example: Students by gender (boy/girl) and sport preference (soccer/basketball).

Joint frequency: The count in a specific cell (e.g., boys who like soccer).

Marginal frequency: The totals in the last row/column.

Relative frequency: A frequency divided by the total, expressed as a fraction, decimal, or percent.

Example: If 15 out of 50 students are boys who like soccer, relative frequency = 15/50 = 30%.

Conditional Relative Frequency

Sometimes you want the frequency within a specific row or column, not of the whole table.

"Of the boys, what percent like soccer?" → Divide boys-who-like-soccer by total-boys.

This is different from "What percent of ALL students are boys who like soccer?"

The first is a conditional relative frequency. The second is a joint relative frequency.

📝 Worked Example: Two-Way Table

In a class of 40: 12 boys play sports, 8 boys dont. 14 girls play sports, 6 girls dont. What fraction of students play sports?

  1. Total who play sports: 12 + 14 = 26.
  2. Total students: 40.
  3. Relative frequency: 26/40 = 65%.
💡 Remember: Joint frequency = one cell. Marginal frequency = row/column total. Relative = cell/total!
